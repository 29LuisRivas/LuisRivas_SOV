Investigación sobre HTML, CSS y JavaScript

HTML (HyperText Markup Language)

HTML es el lenguaje estándar para crear páginas web. Utiliza una serie de elementos para estructurar y presentar contenido en la web, como texto, imágenes, enlaces, etc. Los elementos de HTML se representan mediante etiquetas, que rodean el contenido y definen su propósito en la página.

  • Elementos comunes: <div>, <p>, <a>, <img>
  • Versión más reciente: HTML5

CSS (Cascading Style Sheets)

CSS es un lenguaje de estilos utilizado para describir la presentación de un documento escrito en HTML o XML. CSS permite controlar el diseño visual de las páginas web, separando la estructura del contenido del estilo visual.

  • Propiedades comunes: color, background-color, margin, padding, font-size
  • Posibilidades: Diseño responsivo, animaciones, y control de layouts complejos.

JavaScript

JavaScript es un lenguaje de programación que se utiliza para crear contenido dinámico y interactivo en la web. Permite manipular el DOM (Document Object Model) y responder a eventos como clics de usuario, entradas de teclado, y más.

  • Características: Lenguaje basado en eventos, permite la manipulación de elementos HTML en tiempo real.
  • Uso común: Validación de formularios, animaciones, peticiones AJAX.
